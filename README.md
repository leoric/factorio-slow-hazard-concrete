# Slow Hazard Concrete

Because striped yellow lines mean **SLOW THE F*@# DOWN!**

In vanilla Factorio, hazard concrete serves no real functional purpose other than aesthetics and recycling fodder. This mod adds realistic motion dampening effects to both Hazard Concrete and Refined Hazard Concrete to make them act like actual industrial warning zones.

---

## Features

* **Foot Traffic Dampening:** 
  * Player walking speed is reduced to **60%** on standard Hazard Concrete.
  * Player walking speed is reduced to **30%** on Refined Hazard Concrete.
* **Vehicle Speed Limits:** A low, strict vehicle speed limit is automatically enforced whenever you drive over Hazard or Refined Hazard Concrete.
